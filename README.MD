# Дополнительное задание к Лабораторной работе №4
# Программа сложения трёх знаковых чисел без CALL и RET

## 1. Цель работы
Реализовать комплекс программ (КП) на ассемблере БЭВМ, выполняющий сложение трёх знаковых чисел (`X + Y + Z`) с использованием подпрограммы (ПП).
**Ключевое ограничение:** Команды `CALL` и `RET` **запрещены**. Их необходимо эмулировать с помощью команд `PUSH`, `POP` и `JUMP`.

## 2. Краткое описание алгоритма
Основная программа (П) трижды вызывает одну и ту же подпрограмму (ПП) `FUNC`.
1. Перед вызовом в **стек** помещаются два значения: **адрес возврата** (куда вернуться) и **аргумент** (слагаемое).
2. Выполняется безусловный переход (`JUMP`) на подпрограмму `FUNC`.
3. Подпрограмма извлекает из стека слагаемое, прибавляет его к глобальной переменной `RESULT`, затем извлекает адрес возврата и совершает косвенный переход (`JUMP (TMP)`) обратно в основную программу.
4. Процесс повторяется для чисел `X`, `Y`, `Z`.
5. Результат сохраняется в ячейке `RESULT (0x3FD)`.

## 3. Распределение памяти

| Адрес | Метка | Назначение | Начальное значение |
| :--- | :--- | :--- | :--- |
| `0x3FD` | `RESULT` | Накопитель суммы | `0x1222` (будет перезаписано) |
| `0x3FE` | `X` | Первое слагаемое | `0x1333` (+4915) |
| `0x3FF` | `Y` | Второе слагаемое | `0x0008` (+8) |
| `0x400` | `Z` | Третье слагаемое | `0xFFFF` (-1) |
| `0x401` | `R1` | Адрес возврата 1 | `0x40B` |
| `0x402` | `R2` | Адрес возврата 2 | `0x410` |
| `0x403` | `R3` | Адрес возврата 3 | `0x415` |
| `0x404` | `START` | Точка входа в программу | Код |
| `0x600` | `FUNC` | Подпрограмма суммирования | Код |
| `0x606` | `TMP` | Временная ячейка для адреса | `0x0000` |

## 4. Инструкция по использованию

Нужно воспользоваться компилятором ассемблера БЭВМ затем ввести по соответсвующим адресам соответствующие значения переменных, нажать пуск и следить за работой программы.
